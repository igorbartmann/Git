O que Aprendi:

Neste curso aprendi os conceitos básicos de Git e GitHub e os principais comandos para utilizar o Versionamento de Código e trabalhar com Repositórios Git.

Git:

Git é um Sistema de Controle de Versões (VCS) bastante famoso e atualmente utilizado por grande parte dos desenvolvedores de software.

GitHub:

GitHub é uma plataforma da Microsoft que permite a hospedagem de código-fonte e arquivos com controle de versão através de repositórios Git.
